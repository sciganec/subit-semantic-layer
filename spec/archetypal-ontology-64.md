# SUBIT‑64 Archetypal Ontology  
### Full Canonical Expanded Version (All 64 States)

The SUBIT‑64 ontology defines 64 archetypal semantic states derived from the six binary SUBIT axes.  
Each state is a stable semantic attractor with a distinct cognitive and behavioral signature.

---

## 1. Axes

| Bit | Axis               | 0‑Pole        | 1‑Pole        |
|-----|--------------------|---------------|---------------|
| b1  | Order / Chaos      | Order         | Chaos         |
| b2  | Self / Other       | Self          | Other         |
| b3  | Abstract / Concrete| Abstract      | Concrete      |
| b4  | Active / Passive   | Active        | Passive       |
| b5  | Inner / Outer      | Inner         | Outer         |
| b6  | Flow / Structure   | Flow          | Structure     |
