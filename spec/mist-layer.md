# 4. MIST Reasoning Cycle

The MIST cycle consists of **six canonical steps**:

### **Step 1 — Framing**  
Define the semantic frame based on the archetype and axes.

### **Step 2 — Extraction**  
Identify key elements, signals, and constraints.

### **Step 3 — Structuring**  
Organize extracted elements into a coherent internal structure.

### **Step 4 — Interpretation**  
Apply archetypal logic to derive meaning.

### **Step 5 — Projection**  
Generate possible trajectories, outcomes, or interpretations.

### **Step 6 — Consolidation**  
Stabilize the reasoning into a final, coherent cognitive output.

This cycle runs **once per response**, but can recurse internally.
